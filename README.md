Escort you, always remind you to rest, and don't get too tired.

## Settings

- `vscode-care.enabled`: Enable/disable the extension.
- `vscode-care.workStart`: Work start time (24h), default `9:00`.
- `vscode-care.workEnd`: Work end time (24h), default `18:00`.
- `vscode-care.weekendBlock`: Block typing on weekends (undo + prompt).
- `vscode-care.waterReminder`: Water reminder around `xx:31` when you type.
- `vscode-care.statusAnimation`: Animate status bar text.
- `vscode-care.breakReminder`: Break reminder after continuous coding.
- `vscode-care.breakIntervalMinutes`: Break reminder interval, default `50`.
- `vscode-care.idleResetMinutes`: New session after idle minutes, default `5`.

## Commands

- `vscode-care: Menu` (click status bar)
- `vscode-care: Toggle Enabled`
- `vscode-care: Snooze 10/30/60 Minutes`
- `vscode-care: Resume Notifications`
- `vscode-care: Allow Coding Today`
- `vscode-care: Open Today Report`

Notes:

- `Snooze` and `Allow Coding Today` are persisted across window reloads for the same day.
